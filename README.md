# 📌 Plugin: DevTools-Blocker!

An HFS plugin that increases front-end security by blocking access to browser developer tools.
 
⚙️ Features:

👉 Blocks access to DevTools (F12, Ctrl+Shift+I).

👉 Prevents viewing of source code via Ctrl+U.

👉 Disables the page zoom function (Ctrl + / Ctrl -).
 
🖥️ Prevent users from inspecting or modifying interface elements, ensuring greater control over the website's presentation.

## Installation
1. In HFS, go to **Menu → Plugins → Install from Repo**
2. Type: DevTools-Blocker
3. Activate the plugin in the HFS dashboard.
